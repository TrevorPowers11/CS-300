# CS-300

The main issue that was solved throughout this course was an effective and convenient storage of course data. To achieve that goal, Project One required to analyze different data structures and their efficiency. Then, in Project Two, this information was applied in the creation of an effective and efficient program.

While approaching the task, I made sure to understand the specifics of data structures and why they matter. This knowledge is important since data structures play a key role in program efficiency and the capacity to manage large quantities of data. For this reason, I decided to use a binary search tree in my project. This data structure ensures the highest level of efficiency when sorting and searching for items in the list, which is what this assignment required.

During my work on the assignment, I faced some obstacles, including difficulties associated with the proper implementation of reading a CSV file and compilation of all components of the code. Also, there were situations when my code worked, but its results did not meet expectations. To address these problems, I performed comprehensive tests of each component of my algorithm, reviewed my logical approach, and identified and fixed errors that arose during the development.

Working on this project impacted my future programming practices. Now, apart from the technical side of writing code, I consider its performance and the possibility of scaling my solution up when dealing with much larger quantities of data. Also, before writing my code, I devote more time to its preparation than I used to do earlier.

Overall, this project improved my programming skills. When developing a code, I try to increase its readability and organization by structuring it and dividing it into functions. At the same time, I think about the ease of maintenance and modification of my solution.
